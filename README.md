# Oregon Hikes

##### This app lists hikes in Oregon.
Users can--- 
* View, edit, add new hikes
* Sign up
* Login
* View maps of listed trails(Google Maps)
* See google reviews
* Know the difficulty of the trail
* Delete a trail (Shorty will be removed. Will only be for admin use)
